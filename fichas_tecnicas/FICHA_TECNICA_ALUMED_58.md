# FICHA TÉCNICA — ALUMED 58 COPLANAR
> Serie: Abisagradas — Puertas y escaparates | Fecha ficha: 01-02-2020
> Fuente: Ficha técnica oficial Alumed Sistemas

## Descripción general

Serie abatible especialmente diseñada para puertas de grandes dimensiones y escaparates. Permite cerradura automática con cilindro de seguridad y opción de vaivén. Sistema coplanar de alta resistencia.

## Características técnicas

| Parámetro | Valor |
|---|---|
| Tipo de serie | Abisagrada — Puertas y escaparates |
| Rotura de puente térmico | No |
| Ancho de hoja | 154,6 mm (con marco) |
| Juntas | EPDM + Felpudo Fin-Seal |
| Escuadras | Extrusión de aluminio |
| Apertura | Interior / Exterior |
| Cierre | Cerradura automática con cilindro de seguridad |
